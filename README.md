To install run:

``` bash
curl --proto '=https' --tlsv1.2 https://raw.githubusercontent.com/mateuszradomski/dotfiles/master/setup.sh | bash
```

If you want to add git do:

``` bash
export GIT=true
curl --proto '=https' --tlsv1.2 https://raw.githubusercontent.com/mateuszradomski/dotfiles/master/setup.sh | bash
```
